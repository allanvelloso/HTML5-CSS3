Principais formatações

Negrito / Destaque

Antigamente podia ser usado a tag <b> para destacar em negrito, hoje em dia o mais utilizado para destacar é <strong> O estrong é usado para dar mais significado a frase.

Italico / Enfase

Nesta frase, temos um termo em italico usando a tag <i> (não semantico).

Nesta frase, termos um termo em enfase usando a tag <em>

usado exatamente para dar enfase ao texto.

Texto Marcado

Podemos criar também um marcado usando a tag Mark <mark>, podemos também usar css diretamente na tag por exemplo mesma forma usando:

<mark style="beckgroud-color:cor;"> "sua frase" e fechar a tag mark usando </mark>

Também tem a forma mais usada, utilizando o CSS utilizando o Style no Body , podendo demarcar toda area em mark de uma só vez.

Por exemplo:

< Style >
mark {
background-color:"sua cor"
}

Textos grandes e pequenos

Estamos criando um texto grande e um texto pequeno nesse parágrafo.

Para criar o testo grande <big> e pequeno <small>

Texto Deletedo

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado.

<del>

Texto Inserido

Podemos marcar um texto como inserido para dar uma enfase e indicar que ele foi adicionado depois.

<ins>

Também pode ser usada a tag U para criar sublinhado, mas não está em uso.

<u>

Texto sobrescrito

Para inserir coisas do tipo x20+3

<sup> para ficar em cima

Texto subscrito

Para inserir coisas do tipo H2O

<sub> para ficar em baixo
